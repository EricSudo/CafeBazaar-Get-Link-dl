# CafeBazaar-Get-Link-dl

Learn install
============

- در ابتدا یک پوشه در هاست خود ایجاد کنید
- حالا اطلاعات داخل فایل فشرده دانلود شده را در آن کپی کنید
- حالا فایل کافه بازار را اجرا کنید
